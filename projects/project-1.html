This project was for the Progressive Web Applications course of the minor web design & development. During this course we learned (as the name might suggest) how to design and develop a progressive web application. I learned a lot of new things on this project.

Sometimes, as a student, you just get through it. You have to study, but the concentration and certainly the inspiration is gone. That inspiration can be rekindled through quotes. This web application has the necessary quotes that give the students inspiration to get through the day.

User story
As a digital design student, I want to be able to view inspiring web design quotes, so that I can get a little bit of energy again when I get tired of all those hard-core deadlines that come at me.

Technical info

The webapp is server side rendered with NodeJs, Express & EJS. This makes the webapp very fast. The quotes are fetched (server side) from an external API. For more information I would recommend to take a look at my Github repo.
